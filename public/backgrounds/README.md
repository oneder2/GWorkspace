# 背景图片目录

此目录用于存放项目的背景图片资源。

## 使用说明

1. 将背景图片文件放置在此目录下
2. 支持的图片格式：`.jpg`, `.jpeg`, `.png`, `.webp`
3. 推荐图片尺寸：1920x1080 或更高分辨率
4. 推荐文件大小：小于 500KB（可使用图片压缩工具优化）

## 默认背景

- **文件名**：`default.jpg`
- **路径**：`/backgrounds/default.jpg`
- 在 `src/App.vue` 中引用

## 添加新背景

1. 将图片文件复制到此目录
2. 修改 `src/App.vue` 中的背景图片路径：
   ```vue
   :style="{ backgroundImage: `url('/backgrounds/your-image.jpg')` }"
   ```

## 注意事项

- 图片文件会被包含在构建产物中
- 建议使用压缩后的图片以减小打包体积
- 确保图片文件名不包含空格和特殊字符
