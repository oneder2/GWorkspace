- **友站链接**：添加 YYSUNI 和 ElysiumStack 两个友站链接

- **仿照参考网站样式**：参考 ElysiumStack 博客样式进行优化
- **日期格式优化**：改进日期显示格式为中文格式（如：2025年12月29日 17时19分）
- **详细信息展示**：
  - 添加观看数显示（带图标）

- **布局优化**：重新组织博客条目信息布局，提升可读性
- **视觉改进**：优化信息分隔符和图标样式，使信息层次更清晰

### 工具页面优化
- 修复工具页面 CSS 变量引用问题
- 优化工具选择器的样式一致性

## v2.2.0 (UI/UX优化版本 - 2025-01-XX)

### UI/UX 全面优化

- **背景蒙板优化**：
  - 优化了全局背景图片的遮罩层设计
  - 亮色模式使用浅色渐变（from-white/45 via-white/35 to-white/55），提高文本对比度
  - 暗色模式使用深色渐变（from-slate-900/75 via-slate-800/80 to-slate-900/85），确保浅色文字清晰可读
  - 改进了遮罩层的过渡动画，提供更流畅的视觉体验

- **文本颜色优化**：
  - 修复了HomePage主标题在背景图片上对比度不足的问题（从text-slate-200改为text-slate-800 dark:text-slate-100）
  - 优化了Header标题颜色，确保在所有背景下都有足够的对比度
  - 改进了PortfolioPage和BlogDetailPage中的标题颜色
  - 所有标题现在都符合WCAG对比度标准，提供更好的可读性

- **玻璃效果透明度优化**：
  - 优化了glass-main的背景透明度（从rgba(200, 200, 200, 0.35)改为rgba(255, 255, 255, 0.55)），提高文本可读性
  - 改进了glass-card的透明度设置，平衡背景可见性和文本清晰度
  - 优化了glass-sidebar的透明度，确保导航文本清晰可读
  - 统一了各个玻璃效果类的透明度设计，提供更一致的视觉体验

- **视觉效果改进**：
  - 为主标题添加了drop-shadow效果，增强文本在背景上的可读性
  - 优化了悬停状态的透明度变化，提供更好的交互反馈
  - 改进了暗色模式下的背景遮罩，使用更深的渐变确保对比度

这些改进显著提升了项目的UI/UX质量，确保文本在所有背景下都有足够的对比度和可读性。

## v2.0.0 (2025-01-XX)

- **用户系统**：完整的用户注册/登录功能，JWT Token 认证
- **管理后台**：管理员专用的后台管理系统

  - 数据分析：访问统计、热门文章排行
  - 评论管理：审核、删除、标记垃圾评论
- **数据持久化**：从文件系统迁移到 SQLite 数据库
- **API 重构**：完整的 RESTful API，支持用户关联
- **安全增强**：密码加密、认证中间件、权限控制
- **移除前端编辑入口**：博客编辑功能仅限管理后台
- **管理员位置更新**：前端获取真实公网IP并自动更新管理员位置信息
